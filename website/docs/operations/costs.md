# Costs

Cost estimation and optimization.

## OpenAI API Costs

### gpt-4o-mini (Recommended)

- Input: $0.15 / 1M tokens
- Output: $0.60 / 1M tokens
- Average lesson: ~$0.001-0.003

### gpt-4o

- Input: $2.50 / 1M tokens
- Output: $10.00 / 1M tokens
- Average lesson: ~$0.01-0.03

## Infrastructure Costs

- **Vercel**: Free tier (hobby) or $20/month (pro)
- **Supabase**: Free tier or $25/month (pro)
- **Railway/Render**: $5-20/month

## Optimization Tips

1. Use `gpt-4o-mini` for development
2. Use MOCK mode for testing
3. Optimize prompts to reduce tokens
4. Cache similar content
5. Monitor token usage
